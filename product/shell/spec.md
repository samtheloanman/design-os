# Application Shell Specification

## Overview
A high-fidelity implementation of the custommortgageinc.com shell. Features a two-tier utility/navigation header and an expansive Cyan-based footer.

## Navigation Structure
### Top Utility Bar (Cyan #1DAED4)
- Left: Phone (877-976-5669)
- Right: Sign In, APPLY NOW

### Main Navigation (White)
- Logo: CMRE (Left)
- Links: Residential Mortgage, Commercial Mortgage, NonQM Stated No Income, Hard Money Mortgage Loans, Contact Us
- Right CTA: Request Call Back

## Layout Pattern
- Fixed Header with two distinct rows.
- High Contrast Header Top Bar.
- Multi-column Footer with Cyan (#1DAED4) background and White text.

## Footer Structure
- Column 1: Logo, Tagline, Social Icons (FB, X, LI, IG).
- Column 2: Quick Links (About, Residential, Commercial, Testimonial, Loan Updates, Contact).
- Column 3: Information (Disclaimer, Support, FAQ, Terms, Privacy).
- Column 4: Compliance/Legal icons and NMLS info.

## Design Notes
- Primary Headings: Bebas Neue (Uppercase).
- Body Text: Raleway (Weight 600).
- Accent Color: #1DAED4 (Teal/Cyan).
- Text Color: #1A202C (Dark Slate).
